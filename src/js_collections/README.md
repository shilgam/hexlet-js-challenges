### Паттерн мемоизация
Мемоизация -- сохранение результатов выполнения Функций для предотвращения повторных вычислений.

```
methodName() {
  if (!this.memo) {
    this.memo = ...;
  }
  return this.memo;
}

methodName();
methodName(); // memo
```
